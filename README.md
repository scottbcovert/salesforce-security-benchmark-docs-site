# Security Benchmark for Salesforce (SBS)

This repository contains the source content for the Security Benchmark for Salesforce (SBS), a vendor-neutral compliance standard for Salesforce security posture.

## Documentation

Full documentation is available at [securitybenchmark.dev](https://securitybenchmark.dev)

## Contributing

SBS is a community-driven standard. We welcome contributions of new controls, improvements to existing controls, and documentation enhancements.

See [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines on submitting pull requests.

## Machine-Readable Format

Each release includes an XML representation of all controls suitable for automated scanning tools. See [For Security Vendors](https://securitybenchmark.dev/for-vendors) for integration guidance.
